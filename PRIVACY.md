# Privacy Policy for X-Simplify

**Last updated:** February 24, 2026

## Overview

X-Simplify is a browser extension that simplifies the X.com (Twitter) interface by hiding non-essential UI elements. It is designed with privacy in mind.

## Data Collection

X-Simplify does **not** collect, store, transmit, or share any personal data or browsing information.

## Storage

The extension uses Chrome's local storage API solely to save your enable/disable preference. This data is stored locally on your device and is never sent to any server.

## Permissions

- **storage**: Used only to remember whether the extension is enabled or disabled.
- **host_permissions (x.com, twitter.com)**: Used only to inject CSS that hides UI elements on these sites. No page content is read, collected, or transmitted.

## Third-Party Services

X-Simplify does not use any third-party analytics, tracking, or advertising services.

## Changes

If this policy changes, the update will be posted here.
